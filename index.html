Cara Mengisi

Format Excel yang dibaca ButirCerdas

Sheet Jawaban

Baris pertama wajib berisi header.

  • No, Nama, Kelas, 1, 2, 3, dan seterusnya
  • Kolom nomor soal boleh sampai jumlah soal terakhir
  • Jawaban peserta diisi A-D atau A-E sesuai pilihan

Sheet Kunci

Berisi nomor soal dan kunci jawaban.

  • Header wajib: Nomor dan Kunci
  • Jumlah nomor harus sama dengan jumlah kolom soal
  • Kunci tidak boleh kosong

R Tabel Validitas

Digunakan sebagai pembanding validitas butir.

  • Kosong: sistem memakai kategori korelasi otomatis
  • Diisi: valid bila r hitung lebih besar atau sama dengan r tabel
  • Contoh input: 0.36

Alur Analisis

Data diproses langsung di browser.

  • Download template, isi Excel, lalu upload
  • Periksa preview dan klik Hitung Analisis
  • Export laporan Excel atau PDF setelah hasil muncul

Interpretasi hasil analisis

Tingkat Kesukaran

P = jumlah peserta benar / jumlah peserta.

  • 0.00 - 0.30 = Sukar
  • 0.31 - 0.70 = Sedang
  • 0.71 - 1.00 = Mudah

Daya Pembeda

D = proporsi benar kelompok atas - kelompok bawah.

  • < 0.00 = Buruk
  • 0.00 - 0.19 = Jelek
  • 0.20 - 0.39 = Cukup
  • 0.40 - 0.69 = Baik
  • ≥ 0.70 = Sangat Baik

Validitas

Menggunakan corrected item-total correlation.

  • Jika R tabel diisi, valid bila r hitung ≥ R tabel
  • Jika kosong, kategori memakai kekuatan korelasi

Reliabilitas

KR-20 = k/(k-1) × (1 - Σpq / Var total).

  • < 0.20 = Sangat Rendah
  • 0.20 - 0.39 = Rendah
  • 0.40 - 0.59 = Cukup
  • 0.60 - 0.79 = Tinggi
  • ≥ 0.80 = Sangat Tinggi
