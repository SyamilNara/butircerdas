Format data

Baris pertama adalah header. Kolom pertama boleh berisi nama responden.

  • Contoh: Nama, S1, S2, S3, S4
  • Isi soal hanya boleh 1 atau 0
  • Nama kosong otomatis menjadi Responden A, B, C, dan seterusnya

Arti nilai

Data yang dianalisis adalah skor benar dan salah.

  • 1 = jawaban benar
  • 0 = jawaban salah
  • Tidak menggunakan pilihan A/B/C/D

Analisis 1

Tingkat kesukaran

Tujuan: mengetahui mudah atau sukarnya tiap soal.

Rumus: P = B / N, dengan B = jumlah responden benar dan N = jumlah seluruh responden.

Analisis 2

Daya pembeda

Tujuan: melihat kemampuan soal membedakan kelompok skor tinggi dan kelompok skor rendah.

Rumus: D = (BA / JA) - (BB / JB), dengan BA = benar kelompok atas, BB = benar kelompok bawah, JA/JB = jumlah anggota kelompok.

Analisis 3

Validitas butir

Tujuan: mengetahui hubungan skor tiap butir dengan skor total tes.

Rumus: korelasi Pearson antara skor butir X dan skor total terkoreksi Y, yaitu total skor dikurangi skor butir tersebut.

Catatan: jika R tabel diisi, status valid ditentukan dari r hitung ≥ R tabel.

Analisis 4

Reliabilitas tes

Tujuan: mengetahui konsistensi keseluruhan tes.

Rumus: KR-20 = k/(k-1) × (1 - Σpq / Var total), dengan k = jumlah soal, p = proporsi benar, q = 1-p.
